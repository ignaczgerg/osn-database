Organic Solvent Nanofiltration (OSN) Database

OSN Database is the first repository for organic solvent membrane separation data. The database contains process parameters (temperature, pressure), flux data and rejection values of different membranes, solvents and small molecules. The dataset is available under the Creative Commons Attribution 4.0 International license (CC-BY).

The original data was first released in 2021 within the article titled "Diversity matters: widening the chemical space in organic solvent nanofiltration." The dataset continuously grows as more and more small chemicals are being measured and added to the dataset. Under the latest release, the data contains more than 400 entries.

Currently, the dataset only contains measurements performed in methanol as solvent using polyimide Duramem membranes with different molecular weight cut-offs.

Our aim is to provide high quality data to the membrane community which can be used in advanced modelling tools.
